# Projeto de matemática discreta
- Criptografia RSA
- Backend em C
- Frontend em Python/Tkinter

## Como executar
Usando WSL, MSYS2 ou Linux e tendo instalados o compilador C e o interpretador Python, execute

```shell
$ make build
$ python3 frontend/gui.py
```
